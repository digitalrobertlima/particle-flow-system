Particle Flow System - Versão Simplificada

Esta é uma versão simplificada do nosso sistema de partículas impressionante.

Como funciona: Cada pequeno círculo colorido é uma "partícula" que segue regras simples de física.

Conceitos Demonstrados:

• requestAnimationFrame para animações suaves

• Math.random() para posições e cores aleatórias

• addEventListener para interação com mouse

• canvas para renderização 2D eficiente
